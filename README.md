# WarmPaws – Pet Care in Winter

**WarmPaws** is a modern and responsive web application designed to help furry friends stay **warm, safe, and healthy** during the cold season.  
Users can explore **local pet care services**, **winter pet clothing**, **grooming options**, and **expert tips** — all within a friendly, easy-to-use interface.

---

## Purpose

- The goal of WarmPaws is to provide a one-stop platform for pet owners to ensure their pets’ well-being during winter. From warm outfits to trusted services, everything is tailored to help your pets stay comfortable and cared for.

---

## Key Features

-  **Interactive Home Page** – Welcoming animations and engaging design for a great first impression.  
-  **Pet Care Services** – Explore local pet care and grooming options suited for winter.  
-  **Winter Pet Clothing** – Discover cozy outfits and accessories to keep pets warm.  
-  **Expert Tips & Guidance** – Access seasonal advice and health care articles for pets.  
-  **User Authentication** – Secure login, registration, and password management with Firebase.  
-  **Responsive Design** – Optimized for desktops, tablets, and mobile devices.  
-  **Toast Notifications** – User-friendly feedback with smooth pop-up alerts.  
-  **Smooth Animations** – Powered by AOS and Animate.css for engaging transitions.  
-  **Image Carousel** – Interactive Swiper slider showcasing products and services.

## Technologies and Packages Used

- React – Frontend framework

- React Router DOM – For navigation and routing

- Tailwind CSS – Styling framework

- DaisyUI – Tailwind CSS component library

- Animate.css – CSS animation library

- AOS (Animate On Scroll) – Scroll-based animations

- Lottie React – For animated illustrations

- Firebase – For authentication and data handling

- React Icons – For UI icons

- React Spinners – For loading animations

- React Hot Toast – For toast notifications

- Swiper – For image carousels and sliders

- Vite (Rolldown version) – Build tool and development server
